## 🌟 Overview

**ZEST** is a modern, dark-themed food delivery web application that delivers a premium ordering experience. With **35+ restaurants**, **200+ menu items** spanning Indian, Chinese, Korean, Thai, Mexican, Italian cuisines and more — ZEST is your one-stop destination for any craving.

Built entirely with **vanilla HTML, CSS, and JavaScript** — zero frameworks, zero dependencies, zero build steps. Just open and enjoy.

## ✨ Features

### 🛒 Ordering System
- **35+ Restaurants** across 12+ cuisine categories
- **200+ Menu Items** — Indian, Junk Food, Asian, Italian, Kebab, Sushi & more
- **Item Customizer** — Choose size (Standard / Large), add Extra Cheese or Bacon
- **Real-time Cart** — Slide-in sidebar with live subtotal, tax, and delivery fee calculation
- **Quantity Control** — Increment / decrement items directly in cart

### 🔍 Discovery
- **Instant Search** — Search restaurants, tags, or individual dish names in real time
- **Category Filters** — Filter by Pizza, Indian, Burgers, Sushi, Noodles, Desserts, Drinks & more
- **Restaurant Cards** — Rating, delivery time, delivery fee, and cuisine tags at a glance

### 📍 Location
- **Live GPS Geolocation** — Auto-detect your current location via browser API
- **Manual Address Entry** — Type any delivery address manually
- **Reverse Geocoding** — Converts GPS coordinates to a readable address via OpenStreetMap Nominatim
- **Persistent Storage** — Address saved in `localStorage` across sessions

### 🛵 Order Tracking
- **Animated Live Tracker** — Full-screen tracking overlay with animated rider dot
- **Progress Steps** — Confirmed → Preparing → On the Way → Delivered
- **ETA Circle** — Live countdown timer
- **Rider Info Card** — Rider name, rating, and delivery count
- **Route Animation** — SVG bezier curve with animated progress line

### 🎨 Design
- **Dark Mode Only** — Warm dark palette with saffron-orange accents
- **Glassmorphism Navbar** — Frosted-glass sticky navigation
- **Micro Animations** — Hover effects, cart badge bump, slide-in cart, modal transitions
- **Noise Texture Overlay** — Subtle grain for a premium feel
- **Responsive Design** — Mobile-first, works on all screen sizes

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| **HTML5** | Structure & semantic markup |
| **Vanilla CSS3** | Styling — Grid, Flexbox, custom properties, animations |
| **Vanilla JavaScript (ES6+)** | App logic, state management, DOM manipulation |
| **Google Fonts** | Typography — Bebas Neue, DM Sans, Playfair Display |
| **OpenStreetMap Nominatim** | Free reverse geocoding API |
| **Browser Geolocation API** | Live GPS location detection |
| **localStorage API** | Address persistence |
| **Vercel** | Deployment & hosting |

> ⚡ **Zero npm. Zero build tools. Zero dependencies.**
